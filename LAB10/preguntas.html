¿Por qué es una buena práctica separar el controlador de la vista?

Para matener una interfaz mas amigable y mas legible. Ademas de separar las operaciones, es buena practica dado que al tener capas esta mas estructurado y mejor de entender.

Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?

  • '$SERVER'-> Array que tiene informacion de las rutas y las ubicaciones de script. Su funcion es obtener informacion por medio de entradas que han sido creadas por el servidor web pero no todos las proporcionan.
  • '$FILES'-> Arreglo asociativo de elementos que sube al script por mdeio del metodo POST.
  • '$COOKIE'->Variables que se pasan al script actual por medio de cookies HTTP
  • '$SESSION'-> Contiene variables de sesion.
  • '$REQUEST'->Contiene el contenido de GET,POT y COOKIE
  • '$ENV'-> A traves del metodo del entorno contiene las variables pasadas al script actual

    3. ¿Qué es XSS y cómo se puede prevenir?

    XSS o cross-site scripting es un tipo de vulnerabilidad informatica que se da comunmente en las aplicaciones web. Estos sucede cuando no hay barreras contra los ataques de diferentes tipos de comandos para robar informacion de los usuarios. Esto se puede prevenir por medio de una instalacion de seguridad que este actualizada para que ante uno de estos casos sea bloqueada automaticamente. Tmabien se puede hacer una redireccion a algun sitio de phishing que tiene la proteccion antivirus. Por otor lado se pueden encender las configuraciones personalizadas.
