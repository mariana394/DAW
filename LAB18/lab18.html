¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?
En el desarrollo de RIA's AJAX corre un papel muy importante pues es el encargado de hacer que la interaccion del usuario con el servidor sea fluida, que funcione de manera asincrona, esto es qie sea independiente a la comunicacion con el servidor para asi evitar que en algun momento el usuario este viendo un pagina en blanco. Sin ajax el usuario podria ver una pagina en blanco en el momento en que no se haya ejecutado del todo la peticion y se este demorando cosa que no queremos que suceda.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

  • ¿Qué implicaciones de seguridad tiene AJAX?

    • Vulnerabilidades de SQL inyection, XSS
  • ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

    • Se deben de hacer del lado del servidor porque de este lado se puede tener mayor control sobre lo que esta sucediendo.

    ¿Qué es JSON?
    Es un formato de intercambio de datos, esta basado en JS, Standar ECMA-262 3rd Edition. Es independiente del lenguaje pero utiliza convenciones de los lenguajes C, C++, C#, JAVA, JS, PERL, Python, entre otros. Formato de facil lectura para los programadores y facil de interpretar para las maquinas.
