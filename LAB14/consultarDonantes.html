Preguntas:

¿Qué es ODBC y para qué es útil?

ODS o Open DataBase Connectivity es un sistema de windows que se puede conectar con cualquier base de datos. Es util porque permite acceder a cualquier dato desde cualquier aplicacion sin importar el sistema en la que se encuentren almacenados los datos.

¿Qué es SQL Injection?

Es la accion en donde un hacker utiliza los cuadros de introduccion de datos de una forma para enviar sus propias peticiones, esto se conoce como inyeccion SQL. Permitiendo a los hackers crear,modificar o eliminar datos que han sido guardados en la base de datos.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

  1. Delimitar los valores de las consultas
  2. Parametrizar las consultas SQL
  3. Utilizar una cuenta con permisos restringidos a la base de datos
  4. No mostrar al usuario la informacion de error generada por la base de datos
  5. Rechazar las peticiones con carateres inusuales
  6. verificar los datos que introduce el usuario
