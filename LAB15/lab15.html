¿Por qué es una buena práctica separar el modelo del controlador?

  1. Para poder clasificar la información, la lógica del sistema y la interfaz.
  2. Si el sitema llegara a cambiar en un futuro o ocurre algun error,es mas facil encontrr los archivos a modificar y se ahorra tiempo.

¿Qué es SQL injection y cómo se puede prevenir?
Es la insercion de codigo SQL por medio de los datos de entrada del lado del cliente, por medio de este codigo la persona que esta inyectando este codigo podra modificar las consultas actuales y modificarlas para modificar algo totalmente distinta. Adicional la persona podra acceder a todas las bases de datos alojadas en el mismo servidor.

  • Como prevenirlo:
  • Parametrizar las consultas SQL
  • Usar cuentas con permisos restringidos a la base de datos.
  • Rechazar peticiones con caracteres especiales.
  • Verificar siempre los datos que ingresa el usuario.
