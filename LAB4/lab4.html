¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programacion que es utilizado con un proposito general ya sea para aplicaciones web o de escritorio. Con el se puede realizar cualquier tipo de programa, ademas es un lenguaje independiente de la plataforma lo que lo hace compatible con cualquier ordenador del mercado. En cambio JavaScript es un lenguaje de programacion especifico para el desarrollo web HTML, no es un lenguaje compilado. Es interpretado por el navegador y se ejecuta directamente. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • now(): Regresa la fecha y hora actual
  • parse(): Regresa los milisegundos que han pasado desde 01-01-1970 hasta la fecha actual
  • UTC():Regresa el numero de milisegundos desde la madrugada de 01-01-1970 hasta la fecha con respecto a la hora UTC
  • valueOf(): Regresa los valores primitivos de una fecha
  • Setters y Getters del dia, el año, el mes, los minutos (etc)

    • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • CONCAT: encadenar diferentes elementos que estan separados por comas o que estan en diferentes matrices
  • SLICE: hace una copia de una seccion especifica de una matriz
  • INDEXOF: regresa el indice del primer elemento que coincida con el parametro
  • SORT: Ordena los elemntos en el arreglo.
  • REDUCE: reduce el arreglo en un valor de izquierda a derecha.

    ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Esta variable al ser global puede ser utilizada dentro de todo el archivo JS lo que significa que ninguna otra puede tener el mismo nombre y al poder ser modificada facilmente puede llegar a generar confusion si su uso no es el correcto.
