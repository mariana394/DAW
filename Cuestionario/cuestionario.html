¿Que es entonces una base de datos -DB?

Es una coleccion de archivos que de una u otra manera estan conectados, estos archivos contienen datos especificos que satisfacen las necesidades de la empresa.

¿En qué casos es conveniente usar bases de datos?

  • Cuando hay muchos archivos interrelacionados
  • Cuando hay muchos archivos que son solicitados por muchos usuarios o aplicaciones a la vez
  • Cuando el volumen de datos es muy grande

    • ¿Qué es un sistema de gestión de base de datos - DBMS?

    Un sistema de gestion de base de datos o DBMS es una coleccion de datos que tiene como objetivo principal permitir extraer y almacenar información de la base de datos de una manera eficiente.
